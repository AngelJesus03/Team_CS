Best of Two Codechef

Cheff took an examination two times. In the first attempt, he scored X marks while in the second attempt he scored Y marks.

According to the rules of the examination, the best score out of the two attempts will be considered as the final score.

Determine the final score of the Chef.

Input Format

  • The first line contains a single integer T - the number of the test cases.
  • The first line of each test contains two integers X and Y.

Output Format

For each test case, output the final score of Chef in the examination.

Constraints

  • 1 ≤ T ≤ 1000
  • 0 ≤ X, Y ≤ 100
